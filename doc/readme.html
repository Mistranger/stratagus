What is ALE Clone?

A clone of the realtime fantasy strategy game W*rCr*ft II.
ALE Clone runs under UNIX or BSD with X11 or console or any Win32.

Since ALE Clone doesn't provide graphics, sounds and documentation,
you need the original game so ALE Clone can extract the necessary files. (With over 1 million sold copies this shouldn't be a big problem)

Alternative you could download the free graphic from the
http://fgp.cjb.net - FreeCraft Graphic Project.

The newest version of FreeCraft (formerly known as ALE Clone) could always be found here.
http://FreeCraft.Org - A free fantasy real time strategy game engine.
