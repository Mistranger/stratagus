Installation instructions for Stratagus 2.1

Software Requirements

  • A supported operating system: Win32, Linux, Mac OS X, BSD, Possibly any other Unix/POSIX OS.
    NOTE: some features of Winsock2 (WS2_32.DLL) are used. So even if you plan to play single player only, WinNT4.0 users will need Service Pack 3 or higher and Windows 95 users will need the winsock 2 update from here.

  • SDL (required).

  • libpng (required).

  • zlib (required).

  • libogg/libvorbis (recommended).

  • libbzip2 (optional).

  • libFLAC (optional).

  • libmad (optional).

  • Either the original or expansion wc2 CD, or another data set.
    NOTE: the Battle.net CD edition is not supported.

Hardware Requirements

  • Processor: A 133 MHz CPU (faster recommended).
    NOTE: Stratagus is believed to be endian and word size safe, so other architectures like (big endian) PPC or (64bit) Alpha should also work at least with Linux.

  • Memory: 16 MB of RAM (32 MB for mp3/ogg support).

  • Video Card: Any graphics card that can handle 16 bpp and 640x480 is supported. (OpenGL is supported, and requires a 32Meg Card)

  • Sound Card: Optional. Windows (for Win32) or SDL (for unix or Mac OS X) compatible sound cards are supported.

Installation

Mac OS X users: Extract the "contrib/macosx.tgz" file to the top-level directory and read "Read Me.rtf".

Binary users may skip to step 3.

  • Step 1, Download and Extract:

    • Download the source from here.
      NOTE: both the tar.gz and zip files are the same, we provide both formats for convenience.

    • Unpack the source stratagus-DATE.tar.gz with 
      	tar -zxvf stratagus-DATE.tar.gz
  • Step 2, Compile:

    • GCC/Mingw users: run 'autoconf', then './configure' to create the Makefile. Type './configure --help' to see compilation options. When complete, type 'make depend && make' to compile, and it should compile a working 'stratagus' (or 'stratagus.exe' for win32) binary.

    • MSVC users:
      • Extract the VC++ project and workspace files from stratagus\contrib\msvc.zip into the root stratagus directory.
      • Download the VC++ libraries and extract them into the root stratagus directory.
      • Copy stratagus\contrib\stratagus.ico to stratagus\src.
      • You should now be able to compile stratagus by opening the stratagus.dsw file.

  • Step 3, Extract data:

    Now that you have a working engine, you need some media data before you can actually play the game. You can either use the data from the wc2 CD, or use another data set.

    • Using wc2 data:

      Download Wargus and compile it.

      For Win32, run: 

      	build.bat DIR
      where DIR is the directory to your CD or your installation, (eg, 'e:' or 'c:\war2').

      For Unix, run: 

      	./build.sh -p DIR
      where DIR is the directory to your mounted CD or your installation, (eg, '/cdrom/' or '/mnt/win/war2').

      NOTE: You will need to execute this script from within the root wargus directory or you will run into problems. It will take some time to extract everything.

    • Using another data set:

      You may download another game.

      NOTE you will need to specify the data directory by passing '-d DIR' flag when you run the stratagus binary, by default the binary looks for 'data' in the current directory.

  • Step 4, Run:

    You may now go into the toplevel stratagus dir and run the binary. Remember, if the data dir is not './data', you will need to run stratagus with the '-d' option.
