Software Requirements

  • A supported operating system: Win32, Linux, Mac OS X, BSD, Possibly any other Unix/POSIX OS.
    NOTE: some features of Winsock2 (WS2_32.DLL) are used. So even if you plan to play single player only, WinNT4.0 users will need Service Pack 3 or higher and Windows 95 users will need the winsock 2 update from here.

  • A supported graphics library: SDL, svgalib, X11/Xlib (although X11 is supported, SDL is recommended).
    NOTE: remote displays aren't supported.

  • libpng (required).

  • zlib (required).

  • libbzip2 (optional).

  • zziplib (optional).

  • Either the original or expansion W*rcr*ft 2 CD, or the FcMP data.
    NOTE: the Battle.net CD edition is not supported

Hardware Requirements

  • Processor: A 133 MHz CPU (faster recommended).
    NOTE: FreeCraft is believed to be endian and word size safe, so other architectures like (big endian) PPC or (64bit) Alpha should also work at least with Linux.

  • Memory: 16 MB of RAM (32 MB for mp3/ogg support).

  • Video Card: Any graphics card that can handle 8 bpp and 640x480 is supported.

  • Sound Card: Optional. Windows (for Win32) or SDL (for unix) compatible sound cards are supported.

Installation

Mac OS X users: Extract the "contrib/macosx.tgz" file to the top-level directory and read "Read Me.rtf"

Binary users may skip to step 3

  • Step 1, Download and Extract:

    • Download the source from here.
      NOTE: both the tar.gz and zip files are the same, we provide both formats for convenience.

    • Unpack the source freecraft-DATE.tar.gz with 
      	tar -zxvf freecraft-DATE.tar.gz
  • Step 2, Compile:

    • GCC/Mingw users: run the 'setup' script and answer the questions instructions to create the Makefile. When complete, type 'make depend && make' to compile, and it should compile a working 'freecraft' ('freecraft.exe' for win32) binary. To see all options of setup, type 'setup --help'.

    • MSVC users:
      • Extract the VC++ project and workspace files from freecraft\contrib\msvc.zip into the root freecraft directory.
      • Download the VC++ libraries and extract them into the root freecraft directory.
      • Copy freecraft\contrib\freecraft.ico to freecraft\src.
      • You should now be able to compile freecraft by opening the freecraft.dsw file.

  • Step 3, Extract data:

    Now that you have a working engine, you need some media data before you can actually play the game. There are two options, use the data from the W*rcr*ft 2 CD, or use FcMP.

    • Using W*rcr*ft 2 data:

      For Win32, run: 

      	build.bat DIR
      where DIR is the directory to your CD or your installation, (eg, 'e:\' or 'c:\war2\').

      For Unix, run: 

      	build.sh -p DIR
      where DIR is the directory to your mounted CD or your installation, (eg, '/cdrom/' or '/mnt/win/war2').

      NOTE: You will need to execute this script from withing your root freecraft directory or you will run into problems. It will take some time to extract everything.

    • Using FcMP data:

      Download FcMP from the download page and extract it to the freecraft root directory (where the freecraft binary exists)
      NOTE if you extract to a different directory, you will need to specify the directory by passing '-d DIR' flag when you run the freecraft binary, by default the binary looks for 'data' in the current directory.
