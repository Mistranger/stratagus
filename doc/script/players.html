Stratagus.Players
Type: array
This is the interface to Player information in the game. You can only index it with an integer. There is a max of 16 players right now.
Stratagus.Players[].Allied
Type: bool array
This boolean array represents the allied status of the player. Players[1].Allied[2] is true if player 1 is allied with player 2. This is not reciprocal, if player 1 is allied with 2 it will only stop his units attacking, player 2 doesn't have to do the same.
Stratagus.Players[].BuildingLimit
Type: int
Maximum number of buildings the player is allowed to have.
Stratagus.Players[].Demand
Type: int
Amount of food consumed by the player's units.
Stratagus.Players[].Enemy
Type: bool array
This boolean array represents the enemy status of the player. Players[1].Enemy[2] is true if player 1 is enemy with player 2. This does not have to be reciprocal, see Allied.
Stratagus.Players[].Name
Type: string
The name of the player, as selected by him.
Stratagus.Players[].NumBuildings
Type: int
Total number of buildings for the player
Stratagus.Players[].NumUnits
Type: int
Total number of units for the player
Stratagus.Players[].Score
Type: int
The player's score. FIXME: Document now how this is calculated.
Stratagus.Players[].SharedVision
Type: bool array
This boolean array represents the shared vision status of a player. Players[1].SharedVision[2] is true if player 1 wants to share vision with player 2. Share vision only takes effect if both players agree on it and share vision with each other.
Stratagus.Players[].Slot
Type: int
This is perhaps confusing and rarely usefull. It's the index in the player array. Stratagus.Players[x].Slot == x.
Stratagus.Players[].StartX
Type: int
X coordinate of the player's start location. This only specifies where the screen is placed at the start of the game.
Stratagus.Players[].StartY
Type: int
Y coordinate of the player's start location. This only specifies where the screen is placed at the start of the game.
Stratagus.Players[].Supply
Type: int
Total amount of food supplied for the player.
Stratagus.Players[].TotalBuildings
Type: int
The player's building score. This should be the total number of units built by the player.
Stratagus.Players[].TotalKills
Type: int
How many units the player has killed.
Stratagus.Players[].TotalRazings
Type: int
How many buildings the player has destroyed.
Stratagus.Players[].TotalUnitLimit
Type: int
Maximum number of units and buildings (buildings can be considered units that don't move)
Stratagus.Players[].TotalUnits
Type: int
The player's unit score. This should be the total number of units trained by the player.
Stratagus.Players[].UnitLimit
Type: int
Maximum number of units the player is allowed to have.
Stratagus.Players[].UnitTypesCount
Type: int array
This is the number of units that the player has, organized by type. You can index this by both unit number, or better yet by unit-ident
