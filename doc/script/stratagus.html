Stratagus.GameCycle
Type: int
Game cycle. How many ticks from the start of the game.
Stratagus.GameName
Type: string
The name of the game. Specified ONCE in the config file, it should be unique. It will be the name of the pref dir, and used to as a check in multiplayer. Keep it short and lowercase.
Stratagus.GamePaused
Type: boolean
True if the game is paused, false otherwise.
Stratagus.LibraryFilePath
Type: string
This is the path for the stratagus data dir, the one you specify with -d on the command line.
Stratagus.SyncRand()
Type: function, returns int
This calls the internal stratagus syncronized random number generator. It will get the same values across multiplayer games. Make sure you don't call it from unsincronized parts of the game.
Stratagus.MyRand()
Type: function, returns int
This calls a random number generator. Do not use this in map scripts or anything else that has to behave the same across multiplayer games.
