Q: gzseek already defined or undefined.
A: You have an incompatible libpng version, install the latest version.

Q: The software cursor is too slow.
A: This should be fixed in the latest stable release, if it still happens, compile in SDL video mode or add -DSLOW_INPUT to DFLAGS in Rules.make.

Q: Does FreeCraft work with ALSA.
A: Yes, if you compiled SDL with alsa support and you compiled freecraft using SDL video mode.

Q: Does FreeCraft work with the W*rcr*ft 2 Expansion CD?
A: Yes.

Q: Does FreeCraft work with the W*rcr*ft 2 BNE version?
A: No. The file format of this version is currently not supported.

Q: Can I use FreeCraft to join in a W*rcr*ft 2 network game?
A: Sorry, FreeCraft use its own network protocol. The protocol needs the exact same code executed on all machines, and it is not possible to have this with the original WC2, therefore you can only connect to a FreeCraft server.

Q: What about other RTS games?
A: St*rcr*ft and W*rcr*ft 1 support is in development. T*ber*an Sun may be added in the distant future. For the old C&C games, you can use freecnc.

Q: What ports does the FreeCraft network protocol use?
A: 6660 by default, you may start freecraft with '-Pxxxx' flag to use port xxxx instead.

Q: Does freecraft support internet/metaserver play?
A: Not yet, but this is a planned feature and may be added for 1.19

Q: I get only a black window after I have installed it on windows. What have I done wrong?
A: Install FreeCraft to the C: drive. This has solved the problem for everybody who has had this bug.

Q: I have installed the windows version, when I start FreeCraft I get only the splash screen and the game stops, but consumes 100% CPU. What have I done wrong?
A: Please wait for a couple of minutes for the game to start. There are some machines (Win98 and Win2000) that will still not start after some time. On all machines I have tested it, it works. I have no idea where the problem is, any help is welcome.

Q: My question isn't answered here.
A: Send an e-mail to our hard working support team. :)
